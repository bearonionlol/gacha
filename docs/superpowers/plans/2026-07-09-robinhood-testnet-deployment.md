# Robinhood Testnet Deployment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deploy the Gacha protocol package to Robinhood Chain Testnet, seed sample inventory, smoke-check contract wiring, and prepare the web app to read the deployed registry.

**Architecture:** The contracts package uses Hardhat scripts in `packages/contracts/scripts` and writes deployment state to `deployments/robinhoodTestnet.json`. The web app reads a deployment registry through `NEXT_PUBLIC_GACHA_DEPLOYMENT_REGISTRY` and falls back to deterministic demo state when no registry is provided.

**Tech Stack:** pnpm workspace, Hardhat, ethers v6, Solidity 0.8.28, Next.js app router, Robinhood Chain Testnet chain ID `46630`.

### Task 2: Verify Robinhood Testnet Access

- [ ] **Step 1: Confirm official testnet RPC chain ID**

Run:

```bash
node - <<'NODE'
const https = require("https");
const body = JSON.stringify({ jsonrpc: "2.0", id: 1, method: "eth_chainId", params: [] });
const req = https.request(
  "https://rpc.testnet.chain.robinhood.com",
  { method: "POST", headers: { "content-type": "application/json", "content-length": Buffer.byteLength(body) } },
  (res) => {
    let output = "";
    res.on("data", (chunk) => (output += chunk));
    res.on("end", () => console.log(output));
  }
);
req.on("error", (error) => {
  console.error(error);
  process.exitCode = 1;
});
req.end(body);
NODE
```

Expected: JSON-RPC result is `0xb626`, decimal `46630`.
